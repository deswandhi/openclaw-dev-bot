Yuna AI Assistant

Yuna is a calm, reliable daily work assistant for Data & AI product execution at Indosat. She runs on OpenClaw, replies through Telegram, and uses Alibaba Qwen as the only active LLM provider.

Setup Guide Operations Capabilities

Architecture

The system is intentionally simple: Telegram messages enter OpenClaw, Yuna routes them to Qwen, and the answer returns to Telegram.

Story And Background

Yuna is designed as a steady assistant for high-context product work. She helps convert scattered tasks, meetings, and technical inputs into clear briefs, architecture options, decision notes, and next actions.

The persona is inspired by Final Fantasy X: thoughtful, composed, supportive, and dependable. In production use, that translates into concise answers, careful assumptions, and practical recommendations.

Setup Guide

Environment

cp .env.example .env
chmod 600 .env
nano .env

Set Qwen and Telegram values. Keep OPENCLAW_AGENT_RUNTIME blank for Qwen-only Telegram operation.

Validation

OPENCLAW_CONFIG_PATH=$PWD/openclaw.json openclaw config validate
./scripts/test-qwen.sh
./scripts/test-telegram.sh

Operations Guide

Start

./scripts/start-background.sh

Stop

./scripts/stop-yuna.sh

Restart

./scripts/restart-yuna.sh

Status

./scripts/status-yuna.sh

Logs

./scripts/logs-yuna.sh

Backup

./scripts/backup-config.sh

For auto-start after reboot, use user-level systemd and enable lingering with sudo loginctl enable-linger ubuntu.

Security Best Practices

  • Never commit .env, logs, tokens, or API keys.
  • Keep Telegram pairing enabled.
  • Keep owner commands restricted until an allowlist is configured.
  • Use config backups and review diffs before deployment.

Why Qwen only?

The current production requirement is predictable Qwen-only behavior with no fallback provider.

How is the bot photo updated?

Run ./scripts/update-telegram-profile.sh. If Telegram rejects PNG upload, use BotFather /setuserpic.

Does this page need a server?

No. Open this file directly in a browser.

What if Yuna stops replying?

Check Telegram token, pairing status, Qwen test results, service status, and logs in that order.
